## 📝 작업 규칙

### 우선순위
- **P0**: 즉시 착수, 핵심 기능
- **P1**: P0 완료 후 착수
- **P2**: 추가 기능, 시간 여유 시 진행

### 커밋 메시지
```
feat :: 새로운 기능 추가 #이슈번호
fix :: 버그 수정 #이슈번호
refactor :: 코드 리팩토링 #이슈번호
style :: 스타일 변경 #이슈번호
```

### 브랜치 규칙
```
Feature/#이슈번호-작업내용
Fix/#이슈번호-작업내용
```
